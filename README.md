# 근로감독 자가진단 서비스

고용노동부 근로감독 전, 사업장의 노동법 준수 여부를 스스로 점검할 수 있는 웹 서비스입니다.

## 기능

- **랜딩**: 서비스 소개, 특징, 프로세스, FAQ
- **기본정보 입력**: 사업장명, 업종, 상시근로자 수, 고용형태 등
- **진단**: 7개 카테고리 53문항 체크리스트 (예/아니오/일부/해당없음 등)
- **결과**: 종합 점수, 등급(안전/주의/경고/위험), 카테고리별 점수, 상세 결과, 개선 항목

## 기술 스택

- React 18 + Vite 5
- React Router 6
- Tailwind CSS 3

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속

## 빌드

```bash
npm run build
```

`dist` 폴더에 정적 파일이 생성됩니다.

## 데이터 저장

- MVP에서는 브라우저 로컬 스토리지에 기본정보·진단 응답을 임시 저장합니다.
- 로그인, PDF 다운로드, 히스토리는 향후 버전에서 제공 예정입니다.
